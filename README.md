# AI TOURIST GUIDE

Offline AI system that identifies landmarks from photos and generates tourist-style descriptions.

## V1 Goals (Offline)
- Image → embedding → similarity search → POI
- POI → local data card → guide text
- Future: on-device LLM + TTS

## Repository structure
- `src/ai-guide/` core logic
- `scripts/` runnable scripts
- `data/` local data (not versioned)
- `docs/` documentation

## Requirements
- Python 3.10+
